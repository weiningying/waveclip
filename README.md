# 🎵 音频剪辑器

一个简单的音频剪辑 Web 应用，支持局域网访问。

## 功能

- ✅ 上传音频文件（点击或拖拽）
- ✅ 可视化波形显示
- ✅ 拖拽选择剪辑范围
- ✅ 播放预览片段
- ✅ 导出 MP3（192kbps）

## 快速启动

### 方式一：使用启动脚本（推荐）

```bash
cd /data/audio-editor
./start.sh        # 默认使用 8080 端口
./start.sh 3000   # 指定端口
```

### 方式二：手动启动

```bash
cd /data/audio-editor
python3 -m http.server 8080 --bind 0.0.0.0
```

## 访问方式

启动后会显示访问地址：

- **本机访问**: http://localhost:8080
- **局域网访问**: http://你的IP:8080

同一局域网内的手机、电脑都可以通过局域网地址访问。

## 使用方法

1. 打开页面后，上传音频文件
2. 在波形图上拖动两侧滑块选择剪辑范围
3. 点击「播放片段」预览
4. 点击「导出 MP3」下载剪辑后的文件

## 注意事项

- 所有处理在浏览器本地完成，音频不会上传到服务器
- 推荐使用 Chrome/Edge/Firefox 等现代浏览器
- 大文件处理可能需要一些时间，请耐心等待
